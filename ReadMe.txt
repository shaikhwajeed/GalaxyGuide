Solution build on:
1. Visual Studio 2015
2. .NET Framework 4.5
Note: It should run perfectly fine with other VS versions as long as required framework is installed. 

Assumptions:
1. Units mapping will be provided as initial inputs. ex. "glob is I" will be initial inputs.
2. Input containing numeric value represents the metal credit information.
3. Input string which has question mark is treated as question regardless of other information.

Program accepts Inputs:
1. Command Line Arguments
	a. File Path
	b. Entire input as arguments
2. Console Input
3. Hard-coded input for default test case. ( To save time)
